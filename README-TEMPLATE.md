## Roshambo (aka, "Rock, Paper, Scissors")

### *Tech Used*
* Xcode 8+ (last tested 10.2)
* Swift 3+ (last tested 5.0)
* iOS 8+ (last tested 12.2)

Frameworks:  
- UIKit

### *Description*

Plays the Roshambo, or commonly called "Rock, Paper, Scissors", game, demonstrating the use of the 3 common ways to seque between views.


### *Setup Requirements*

Nothing special - just clone and go!
